## Key Operational Facts

- Model selection in agent systems is a **role assignment problem**, not a capability/cost tradeoff.
- Opus for orchestration, Sonnet for teammates, Haiku for high-volume leaf tasks is the baseline.
- Wrong model assignment at the teammate level compounds: a team of 5 Opus agents costs ~15x a team of 5 Haiku agents for equivalent throughput.
